# NBA数据分析系统 - 模块类图

## 一、用户管理模块

用户管理模块负责用户认证、用户信息管理和用户统计数据管理。

```plantuml
@startuml
left to right direction
skinparam backgroundColor #FEFEFE
skinparam defaultFontName Microsoft YaHei
skinparam classAttributeIconSize 0

class AuthController {
    +login()
    +register()
    +logout()
    +getUserInfo()
    +updateUserInfo()
    +updatePassword()
}

class Emp {
    -id: Integer
    -username: String
    -password: String
    -name: String
    -phone: String
    -image: String
}

class UserStats {
    -username: String
    -viewCount: Integer
    -favoriteCount: Integer
    -predictionCount: Integer
    -favoriteTeam: String
    -favoritePlayer: String
}

class EmpMapper {
    +findByUsername()
    +findById()
    +insert()
    +update()
    +updatePassword()
}

class UserStatsMapper {
    +findByUsername()
    +insert()
    +updateFavorites()
    +incrementViewCount()
}

AuthController --> EmpMapper
AuthController --> UserStatsMapper
EmpMapper --> Emp
UserStatsMapper --> UserStats

@enduml
```

### 类说明

| 类名 | 职责 |
|------|------|
| AuthController | 处理用户认证相关请求，包括登录、注册、登出等 |
| Emp | 用户实体类，存储用户基本信息 |
| UserStats | 用户统计数据实体类，记录用户行为统计 |
| EmpMapper | 用户数据访问层，操作用户表 |
| UserStatsMapper | 用户统计数据访问层，操作统计表 |

## 模块依赖关系总览

```plantuml
@startuml
left to right direction
skinparam backgroundColor #FEFEFE
skinparam defaultFontName Microsoft YaHei

package "用户管理模块" {
    [AuthController] as UC
    [EmpMapper] as UM
}

package "数据查询模块" {
    [DataController] as DC
    [DataService] as DS
}

package "数据可视化模块" {
    [AdvancedStatsController] as AC
}

package "数据对比模块" {
    [UserTeamController] as TC
    [UserLineupController] as LC
}

package "AI分析模块" {
    [AIController] as AI
    [SearchService] as SS
}

package "比赛预测模块" {
    [PredictionController] as PC
    [BasketballPredictionService] as PS
}

UC --> UM : 用户认证
DC --> DS : 数据查询
AI --> SS : 联网搜索
PC --> PS : 比赛预测
PS ..> DS : 获取球队数据

@enduml
```
